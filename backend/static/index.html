Quick Start

  1. Ingest — загрузить C++/Markdown проект в граф
  2. Index Embeddings — сгенерировать векторные представления
  3. Generate Summaries — LLM описания для каждого узла
  4. Vector Search — найти стартовый узел по промпту
  5. Subgraph Explorer — навигация по графу от узла
  6. View Code — полный исходный код узла

Ingest — Загрузка проекта

Папка должна быть примонтирована в docker-compose.yml (volumes). По умолчанию: test_data → /app/test_data

Index Embeddings

Генерирует векторные представления всех узлов через Gemini Embedding API и сохраняет в Qdrant.

Generate Summaries

LLM генерирует краткие описания для каждого узла (при индексации, НЕ при навигации).
