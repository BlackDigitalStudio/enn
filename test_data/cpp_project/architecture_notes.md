# Архитектурные заметки: Renderer

## Связь с физикой

Класс `Renderer` получает ссылки на `RigidBody` и `PhysicsWorld`
для отрисовки. Он не владеет объектами, только читает их состояние.

## Текущие ограничения

`drawBody` рисует только круги (по радиусу из RigidBody).
Для полигонов нужен будет новый тип — PolygonBody.

## Точка входа

Функция `main` в main.cpp создаёт PhysicsWorld, добавляет тела
через `addBody`, и в цикле вызывает `step` + `drawWorld`.
